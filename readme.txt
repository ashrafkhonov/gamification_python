Суть игры
Игроки пишут какую-нибудь простую программу. Каждый игрок по очереди пишет кусок кода. Игрок продолжает писать код с места, на котором остановился предыдущий игрок (по циклу). Во время игры будет вестись учет количества строк, написанных каждым игроком. После добавления фрагмента кода сервер запускает написанный код в компиляторе для проверки его валидности. Игра завершается, если один из игроков допустит ошибку в коде или выйдет из игры.
